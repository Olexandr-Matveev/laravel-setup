## Структура

- `01_cookie/` — форма імені, збереження cookie на 7 днів, видалення cookie.
- `02_session/` — авторизація через сесію, перевірка активної сесії, вихід.
- `03_server/` — виведення `$_SERVER`, перенаправлення при не-POST запиті.
- `04_cart/` — корзина, товари у сесії, попередні покупки в cookie.
- `05_activity/` — автоматичне завершення сесії після 5 хвилин неактивності.
