# Лабораторна робота №3 — ООП у PHP

## Тема
Об'єктно-орієнтоване програмування в PHP.

## Реалізовано
- `AccountInterface` з методами `deposit`, `withdraw`, `getBalance`.
- `BankAccount`, який реалізує інтерфейс.
- Константа `MIN_BALANCE`.
- Властивості `balance` і `currency`.
- Перевірка некоректних сум.
- Виняток `Exception` при недостатній кількості коштів.
- `SavingsAccount`, який успадковує `BankAccount`.
- Статична властивість `interestRate`.
- Метод `applyInterest()`.
- Клієнтський код у `index.php`.

## Запуск
Відкрити термінал у папці `Lab3` і виконати:

```powershell
C:\php\php.exe -S localhost:8000
```

Після запуску відкрити в браузері:

```text
http://localhost:8000/index.php
```

## Критерій успіху
Сторінка відкривається без фатальних помилок. У таблиці видно успішні операції та оброблені винятки:
- поповнення рахунку;
- зняття коштів;
- спроба зняти більше, ніж є на балансі;
- спроба поповнити рахунок від'ємною сумою;
- застосування відсоткової ставки для накопичувального рахунку.
